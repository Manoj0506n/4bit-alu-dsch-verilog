# 4-Bit ALU using DSCH and Verilog HDL

## Overview
This project implements a 4-bit Arithmetic Logic Unit (ALU) using DSCH and Verilog HDL.

## Supported Operations
- Addition
- Subtraction
- AND
- OR
- XOR

## Components Used
- 1-Bit Full Adder
- 4x1 Multiplexer
- Logic Unit
- 4-Bit Ripple Carry Adder

## Tools
- DSCH
- Verilog HDL

- DSCH Schematics
- Verilog Source Code
- Circuit Screenshots
